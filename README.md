# Vortex - Data Layer
Data layer implementation for Unity

[![openupm](https://img.shields.io/npm/v/com.rhinox.open.vortex?label=openupm&registry_uri=https://package.openupm.com)](https://openupm.com/packages/com.rhinox.open.vortex/)
